Since the project is in Farsi, I explain Farsi.

تو این پروژه که میخواستم بوتسترپ رو تمرین کنم، شکل اولیه‌ی یک سایت فروش لباس رو با بوتسترپ ساختم. البته میخوام در آینده بهش جاوا اسکریپت بیشتری اضافه کنم که تعامل همین نمونه‌ی اولیه هم بهتر از حال حاضر باشه. هفت تا صفحه داره: صفحه‌ی خانه، ساخت حساب کاربری، ورود به حساب کاربری، دیدن محصولات یک دسته‌ی خاص، دیدن یک محصول، سبد خرید و ورود اطلاعات خرید.


در صورتی که تمایل دارید از این کدها تو پروژه‌تون استفاده کنید حتما قبلش از من اجازه بگیرید لطفا.
